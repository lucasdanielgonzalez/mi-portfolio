Sobre mí

Analista de Sistemas egresado del Instituto Tecnologico Beltran. Mi formación académica me brindó bases en lógica de programación, bases de datos y desarrollo web. Tras recibirme, especialicé mi perfil en el ecosistema Backend mediante una certificación intensiva en Python y Django.

Actualmente, me enfoco en la arquitectura de servidores y la gestión de datos, utilizando Git y GitHub para un flujo de trabajo profesional.

Mi objetivo es seguir escalando mis habilidades tanto en el Back como en el Front, buscando soluciones integrales que aporten valor real a equipos de desarrollo.
